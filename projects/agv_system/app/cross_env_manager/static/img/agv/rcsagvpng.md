## 1. 素材概述

该图片是一张典型的 UI 图标素材表（Sprite Sheet / Atlas），用于游戏、工业控制界面、物流系统、自动化平台、模拟经营类游戏或编辑器工具界面。

特点：

* 统一蓝灰色线条风格
* 多状态图标（正常 / 激活 / 禁用 / 高亮）
* 适合切片后用于 UI 系统
* 部分图标存在“带标签版本”
* 存在彩色按钮与连接器类素材
* 黑色区域为留白或未使用区域

整张图片分辨率：

* 1280 × 1280 px

# 2. 网格与切片规则

## 基础网格

该素材整体采用：

* 基础单元：128 × 128 px
* 图标间距：约 8~16 px 黑色分隔
* 大部分图标位于标准网格中

建议切片方式：

| 项目     | 数值                     |
| ------ | ---------------------- |
| 单格尺寸   | 128 × 128 px           |
| 推荐切片尺寸 | 96 × 96 或 112 × 112 px |
| 推荐内边距  | 8~12 px                |
| 锚点     | 中心                     |
| 背景     | 透明/黑底兼容                |

# 11. 推荐切片命名规范

```text

```

例如：

```text

power_error
power_disabled
```

# 13. Unity 使用建议

## 导入设置

| 设置项          | 建议                 |
| ------------ | ------------------ |
| Texture Type | Sprite (2D and UI) |
| Sprite Mode  | Multiple           |
| Mesh Type    | Full Rect          |
| Filter Mode  | Point 或 Bilinear   |
| Compression  | None               |

---

## Sprite Editor

建议：

* 自动网格切片
* Cell Size：128 × 128
* Offset：0
* Padding：0~4

# 17. 最佳实践

推荐：

* 使用统一描边颜色
* 不要混入高饱和写实图标
* UI 背景尽量深色
* 图标尺寸保持一致
* 状态变化只改颜色不改轮廓

---

# 18. 总结

这是一个高度模块化的工业/自动化风格 UI 图标素材表。

核心特点：

* 强状态表达
* 明确颜色编码
* 适合大型系统界面
* 支持工具栏与设施混合使用
* 非常适合：

  * 工厂游戏
  * 自动化管理
  * 基地建设
  * 仓储物流
  * 科幻控制台

如果继续扩展，建议：

* 增加 Hover/Pressed 状态
* 提供 SVG 矢量版本
* 增加 2x / 4x 高清导出
* 补充动画帧
* 建立统一 Icon Font 或 UI Theme
