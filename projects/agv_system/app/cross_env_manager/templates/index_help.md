# 跨环境任务模板管理系统 · 使用帮助

## 1. 搜索任务模板

在搜索框中输入任务模板代码或名称，支持模糊匹配。例如：

- 输入 `HJBY` 查找所有包含 HJBY 的任务模板
- 输入完整代码如 `HJBY_back_32A3DJ2F_to_31A3QD4B3F_446`
- 输入部分关键词如 `back` 或 `test`
- 输入纯数字（如 `327`）可直接通过模板ID精确查找

## 2. 复制模板

在搜索结果页面或模板详情页面，点击"复制模板"按钮。

系统会自动生成新的模板代码：
- 如果输入 `HJBY_test`，系统会检查最后一条记录ID（如484）
- 自动生成新ID为485，新模板代码为 `HJBY_test_485`

## 3. 修改模板信息

在模板详情页面点击"编辑"按钮，进入编辑页面。

修改完成后点击"保存"，系统会提示确认修改。

> ⚠️ 注意：修改操作会直接更新数据库，请谨慎操作。

## 4. 子任务管理

在模板详情页面可以：
- 查看所有子任务列表
- 编辑子任务详细信息（任务顺序、模板代码、服务器地址、目标点等）
- 添加新的子任务
- 删除子任务
- 拖拽调整子任务顺序

## 5. AGV查询功能

系统提供完整的AGV任务查询功能：

- **任务查询**：查询AGV任务执行状态
- **设备验证**：验证AGV设备配置和状态
- **模板查询**：查询跨环境任务模板
- **AGV状态**：监控AGV实时运行状态
- **交接点查询**：查询AGV交接点信息
- **货架模型查询**：查询货架模型配置

点击首页查询功能按钮或访问查询功能主页使用所有功能。
