# 共享组件

## 组件列表
| 组件 | 文件 | 说明 |
|------|------|------|
| 导航栏 | navigation.html | 顶部导航栏 |
| 页脚 | footer.html | 页面底部信息 |
| 头部 | header.html | 页面标题区 |
| 卡片 | card.html | 通用卡片容器 |
| 按钮 | buttons.html | 按钮组件库 |
| 表单 | form_section.html | 表单区域组件 |
| 提示 | alerts.html | 消息提示组件 |
| 主题切换 | theme_switcher.html | 暗黑/亮色主题切换 |
| CSS变量 | css_variables.html | 共享CSS变量定义 |

## 使用方式
在模板中通过 `{% include "components/xxx.html" %}` 引用。
