使用说明

在搜索框中输入任务模板代码或名称，支持模糊匹配。例如：

  • 输入 HJBY 查找所有包含 HJBY 的任务模板
  • 输入完整代码如 HJBY_back_32A3DJ2F_to_31A3QD4B3F_446
  • 输入部分关键词如 back 或 test

在搜索结果页面或模板详情页面，点击"复制模板"按钮。

系统会自动生成新的模板代码：

  • 如果输入 HJBY_test，系统会检查最后一条记录ID（如484）
  • 自动生成新ID为485，新模板代码为 HJBY_test_485

在模板详情页面点击"编辑"按钮，进入编辑页面。

修改完成后点击"保存"，系统会提示确认修改。

注意：修改操作会直接更新数据库，请谨慎操作。

数据库表结构
fy_cross_model_process (跨环境任务模板主表)
  • id - 主键ID
  • model_process_code - 模板代码 (如 HJBY_back_32A3DJ2F_to_31A3QD4B3F_446)
  • model_process_name - 模板名称
  • enable - 是否启用 (0=禁用/跨环境任务, 1=启用)
  • request_url - 回调URL
  • capacity - 容量管控值
  • target_points - 目标点位
  • area_id - 区域ID
  • target_points_ip - 目标服务器IP
fy_cross_model_process_detail (跨环境子任务模板明细表)
  • id - 主键ID
  • model_process_id - 关联主模板ID
  • task_seq - 任务顺序
  • task_servicec - 下发服务器地址
  • template_code - 子任务模板代码
  • template_name - 子任务名称
  • task_path - 目标点标识
